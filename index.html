CS Survey System / Booking App

Full-Stack Web Ecosystem | C# ASP.NET, HTML, JavaScript, SQL

A comprehensive full-stack web application ecosystem consisting of a digital survey system and an integrated booking platform. The backend is built with C# ASP.NET, providing robust API endpoints, database management via SQL Server, and authentication services. The frontend utilizes HTML, CSS, and vanilla JavaScript to deliver responsive user interfaces for both survey creation/analysis and appointment booking workflows. Features include real-time data validation, dynamic form generation, administrative dashboards, and RESTful API integration.

Meteor Client (Fork) & Lucid Addon

Minecraft Utility Mods | Java, Minecraft Forge/Fabric

A forked and enhanced version of the Meteor Client, along with the Lucid addon, both developed in Java for Minecraft. These utility mods focus on performance optimization, quality-of-life improvements, and enhanced chat functionalities. The Lucid addon extends the base Meteor Client with additional modules for automation, visual enhancements, and multiplayer communication features. Development involved deep understanding of Minecraft's internal APIs, event-driven programming, and efficient memory management for real-time game modifications.
